## 安装指导
### 安装位置
该产品为扩声麦克风，具有波束功能，安装位置需要满足以下条件：
1.	需要在实际场景的讲话人活动范围中心进行吊装。
2.	吊装高度需要高于场景中的音箱高度。
3.	吊装时保持产品麦克风收音面朝地，并保持产品与地面平行。
4.	将产品收音角度方向朝向讲话人侧的墙壁。
